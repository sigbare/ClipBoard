# P2P File Share & Clipboard Sync

Peer-to-peer application for file sharing and clipboard synchronization between Windows and Linux.

## Features

- File transfer of any size
- Clipboard text synchronization (Ctrl+C)
- Direct P2P connection between devices
- Cross-platform (Windows and Linux)
- Unicode support (Russian, Chinese, emojis)

## Installation

### Windows
Download `P2PFileShare_Windows_x64_v2.6.0.zip`, extract and run `ClipBoard.exe`

### Linux
Download `P2PFileShare_Linux_x64_v2.6.0.zip`, extract and run:
```bash
chmod +x ClipBoard
./ClipBoard
